A list of my projects so far!

My litmus test for showing a project here is that I have to be able to say: "this is pretty cool"

  1. Single Bus Non Pipelined Control Unit
    2. This project was assigned as a part of Intro to Computer Architecture & Organization (EECE3026). We were given a set of 16 instructions loosely based on the IBM370 and MIPS instruction sets and tasked to design and implement a CPU for them at the gate level using a single bus paradigm . Logisim was used to implement the state machine for the control unit at the gate level.

  2. Pipelined Control Unit
    3. This project was an extension of the Non-Pipelined Control Unit project. Using the same instruction set, we now had to design a datapath and control that would execute the instructions in a pipelined manner. The big challenge of this project was designing the datapath so as to avoid the many data dependencies/hazards that executing the instructions presented.

  3. Adventure Time Quote Bot
    4. This is a quotebot using Python's Twython package. It tweets a random quote from a list of hundreds of adventure time quotes when run. I typically have it run on a Raspberry Pi 3 using a cronjob. Example bot code can be found on my github account.
